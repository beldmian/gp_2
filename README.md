# ГП-2

## Описание данных

### Основные файлы с вакансиями

#### `data/hh.csv`
- `id` — ID вакансии на HH.
- `url` — ссылка на вакансию.
- `name` — название вакансии.
- `salary` — зарплата в том виде, как она указана в объявлении.
- `description` — текст вакансии.
- `skills` — список ключевых навыков.
- `company` — название компании.
- `address` — адрес работодателя.
- `area` — город.
- `experience` — требуемый опыт.
- `employment` — тип занятости.
- `schedule` — график работы.
- `working_hours` — продолжительность или режим рабочего дня.
- `work_format` — формат работы: офис, удаленно или гибридно.
- `published_at` — дата публикации вакансии.
- `archived` — признак того, что вакансия находится в архиве.
- `source` — источник данных.

#### `data/hh_selenium.csv`
Сокращенная selenium-версия выгрузки с HH.

- `id` — ID вакансии на HH.
- `url` — ссылка на вакансию.
- `name` — название вакансии.
- `salary` — зарплата в том виде, как она показана на странице.
- `description` — текст вакансии.
- `skills` — список ключевых навыков.
- `company` — название компании.
- `area` — город или строка локации.
- `published_at` — дата публикации, если она нашлась.
- `source` — источник данных.
- `query` — поисковый запрос, по которому вакансия была найдена.

> ОФФТОП: Почему selenium в данной задаче избыточен и бесполезен (да и в целом даже вреден)
> Начнем с того что selenium (как и все headless браузеры) нужен в реальных задачах сбора данных
> крайне редко, посколько это дорого (очень дорого если речь идет о повторяющейся задаче сбора).
> Во-вторых в ситуации когда весь трафик можно замаскировать в достаточной степени, чтобы тебе не
> забанили напрочь все прокси / всю систему сбора — надо просто это сделать.)

#### `data/superjob.csv`
- `id` — ID вакансии на SuperJob.
- `url` — ссылка на вакансию.
- `name` — название вакансии.
- `salary` — зарплата в общем виде.
- `salary_from` — нижняя граница зарплаты.
- `salary_to` — верхняя граница зарплаты.
- `currency` — валюта зарплаты.
- `description` — текст вакансии.
- `skills` — ключевые навыки.
- `company` — название компании.
- `address` — адрес работодателя.
- `area` — город.
- `experience` — требуемый опыт.
- `employment` — тип занятости.
- `schedule` — график работы.
- `working_hours` — рабочие часы.
- `work_format` — формат работы.
- `published_at` — дата публикации вакансии.
- `catalogue_ids` — ID профессиональных категорий.
- `catalogue_titles` — названия профессиональных категорий.
- `source` — источник данных.
- `query` — поисковый запрос, по которому вакансия была найдена.

#### `data/superjob_details.csv`
Расширенная версия вакансий SuperJob. Содержит основные поля вакансии, а также дополнительные сведения:

- `company_url` — сайт компании.
- `company_description` — краткое описание компании.
- `education` — требования к образованию.
- `phones` — список контактных телефонов.
- `phone_count` — количество телефонов.
- `metro` — список станций метро.
- `metro_count` — количество станций метро.
- `is_closed` — признак, что вакансия закрыта.
- `is_archive` — признак, что вакансия перенесена в архив.
- `date_published` — дата публикации.
- `date_pub_to` — дата, до которой вакансия активна.
- `latitude`, `longitude` — координаты работодателя.
- `id_client` — ID работодателя в системе.

### Справочные файлы

#### `data/superjob_catalogues.csv`
- `catalogue_id` — ID категории.
- `catalogue_title` — название категории.
- `position_id` — ID подкатегории.
- `position_title` — название подкатегории.

#### `data/superjob_regions.csv`
- `country_id`, `country_title` — ID и название страны.
- `region_id`, `region_title` — ID и название региона.
- `town_id`, `town_title` — ID и название города.

#### `data/superjob_towns.csv`
- `id` — ID города.
- `id_region` — ID региона.
- `id_country` — ID страны.
- `title` — название города.
- `title_eng` — название города на английском.

#### `data/superjob_metro.csv`
- `town_id` — ID города.
- `line_id` — ID линии метро.
- `line_title` — название линии метро.
- `line_color` — цвет линии метро.
- `station_id` — ID станции метро.
- `station_title` — название станции метро.
